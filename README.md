# Go

## Базовый Go-проект для дальнейшей разработки Telegram-бота.

Репозиторий содержит инициализированный Go-модуль, структуру проекта и все необходимые зависимости.

Настроено структурированное логирование с использованием go.uber.org/zap. Логгер инициализирован для режимов development (консоль) и production (JSON) в папке internal/logger/logger.go

## Требования

- Go 1.21 или выше
